Política de Privacidad

Anota Juegos

Última actualización: 27 de abril de 2026

Esta aplicación respeta la privacidad de los usuarios y se compromete a protegerla. A continuación, se describe cómo se maneja la información.

Información que recopilamos

Esta aplicación no recopila información personal identificable directamente, como nombre, dirección de correo electrónico o datos de contacto.

Sin embargo, la aplicación utiliza servicios de terceros que pueden recopilar información automáticamente, incluyendo:

  • Dirección IP
  • Identificadores de dispositivo
  • Datos de uso de la aplicación

Publicidad

La aplicación utiliza Google AdMob para mostrar anuncios. Este servicio puede recopilar y utilizar datos con el fin de personalizar anuncios y mejorar la experiencia del usuario.

Para más información sobre cómo Google maneja los datos, se puede consultar su política de privacidad en el siguiente enlace:

https://policies.google.com/privacy

Uso de la información

La información recopilada por servicios de terceros se utiliza para:

  • Mostrar anuncios
  • Mejorar la experiencia del usuario

Compartición de datos

Esta aplicación no comparte información personal del usuario directamente. Sin embargo, los servicios de terceros utilizados pueden procesar datos conforme a sus propias políticas de privacidad.

Seguridad

Se adoptan medidas razonables para proteger la información del usuario. No obstante, ningún método de transmisión por Internet o almacenamiento electrónico es completamente seguro.

Derechos del usuario

El usuario puede limitar el seguimiento publicitario desde la configuración de su dispositivo móvil.

Cambios en esta política

Esta política puede actualizarse ocasionalmente. Se recomienda revisarla periódicamente para estar informado sobre cualquier cambio.
